# URL-Shortener
A simple full-stack URL shortener application built using HTML, CSS, JavaScript, Node.js, and Express.js.
It allows users to convert long URLs into short, easy-to-share links and redirects them to the original URL.
 
 ->Features
🔗 Shorten long URLs into compact links
⚡ Fast redirection to original URLs
🧠 Simple and clean UI
📦 Backend built with Node.js and Express.js
💾 Lightweight and easy to run locally

->Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Node.js, Express.js
